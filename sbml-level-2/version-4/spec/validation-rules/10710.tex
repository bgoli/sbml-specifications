The value of the \token{sboTerm} attribute on an \Event \changed{should} be
an SBO identifier referring to a mathematical expression.  That is, the
value \changed{should} be a term derived from \sbointeractionID,
\changed{``interaction''}.  (References: L2V2 Section~4.14.1; L2V3
Section~4.14.1; L2V4 Section~\ref{sec:event-sboterm}.)
