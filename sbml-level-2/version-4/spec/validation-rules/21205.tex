The order of subelements within an \Event object instance must be
the following: \token{trigger}, \token{delay},
\token{listOfEventAssignments}.  The \token{delay} element is
optional, but if present, must follow \token{trigger}.
(References: L2V2 Section~4.14; L2V3 Section~4.14; L2V4 Section~\ref{sec:events}.)
