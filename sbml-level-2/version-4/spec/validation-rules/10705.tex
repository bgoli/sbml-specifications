The value of the \token{sboTerm} attribute on a \AlgebraicRule, \RateRule or
\AssignmentRule \changed{should} be an SBO identifier referring to a mathematical
expression.  That is, the value \changed{should} be a term derived from
\sbomathformula.  (References: L2V2 Section~4.11.1;
L2V3 Section~4.11.1; L2V4 Section~\ref{sec:rule-sboterm}.)
