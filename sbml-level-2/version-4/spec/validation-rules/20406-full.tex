Redefinitions of the \changed{predefined SBML} unit \token{volume} must be based on
the unit \token{litre}, \token{metre} or \token{dimensionless}.
More formally, a \UnitDefinition for \token{volume} must simplify to a
single \Unit in which either (a) the \token{kind}
attribute value is \val{litre} and the \token{exponent} has a value of
\val{1}; (b) the \token{kind} attribute has a value of
\val{metre} and the the \token{exponent} has a value of \val{3}, or (c)the
\token{kind} attribute has a value of \val{dimensionless} with
any \token{exponent} value.  (References: L2V1 Section 4.4.3; L2V2
Section~4.4.3; L2V3 Section~\ref{sec:built-in-units}.)
