The \token{units} in a \Parameter definition must be a value chosen from
among the following: a value from the \primtype{UnitKind} enumeration (e.g.,
\val{litre}, \val{mole}, \val{metre}, etc.), a built-in unit (e.g.,
\val{substance}, \val{time}, etc.), or the identifier of a
\UnitDefinition in the model.  (References: L2V1 Section 4.7.3; L2V2
Section~\ref{sec:parameter-units}.)
