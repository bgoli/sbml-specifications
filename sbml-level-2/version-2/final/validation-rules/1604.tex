The value of a \KineticLaw's \token{substanceUnits} can only be one of the
following: \val{substance}, \val{mole}, \val{item}, \val{gram},
\val{kilogram}, \val{dimensionless}, or the identifier of a \UnitDefinition
derived from either \val{mole} (with an \token{exponent} value of \val{1}),
\val{item} (with an \token{exponent} value of \val{1}), \val{gram} (with an
\token{exponent} value of \val{1}), \val{kilogram} (with an
\token{exponent} value of \val{1}), or \val{dimensionless}.  (References:
L2V1 Section 4.9.7; L2V2 Section~\ref{subsec:kinetic-law}.)
