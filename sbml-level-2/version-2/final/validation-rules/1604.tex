The value of a \KineticLaw's \token{substanceUnits} must be either
\val{moles}, \val{substance}, \val{item}, \val{kilo\-gram},
\val{dimensionless}, or the identifier of a \UnitDefinition derived from
either \token{mole} (with an \token{exponent} value of \val{1}),
\token{item} (with an \token{exponent} value of \val{1}),
\token{kilogram} (with an \token{exponent} value of \val{1}), or
\val{dimensionless}.  (References: L2V1 Section 4.9.7; L2V2
Section~\ref{subsec:kinetic-law}.)
