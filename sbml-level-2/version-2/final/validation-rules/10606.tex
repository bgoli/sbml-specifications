If a \Species is located in a \Compartment whose \token{spatialDimensions}
has value \val{2}, then that \Species definition can only set
\token{spatialSizeUnits} to a value of \val{area}, 
\val{dimensionless}, or the identifier of a \UnitDefinition derived from
either \token{metre} (with an \token{exponent} value of \val{2}) or
\val{dimensionless}.  (References: L2V1 Section 4.6.4; L2V2
Section~\ref{sec:species-units}.)
