The mathematical formula in the \token{lambda} expression of a
\FunctionDefinition definition may only refer to variables that have been
declared in the \token{lambda}'s list of function parameters (that is, the
list of bound variables declared using MathML \token{bvar}s).
(References: L2V1 Section~3.6.3; L2V2
Section~\ref{sec:function-definition-math}.)
