When the \token{variable} \changed{attribute} of an \EventAssignment
contains the identifier of a \Parameter in the model, the
units of the mathematical expression in the
\EventAssignment's \token{math} expression must be
consistent with the units declared for that parameter.
(References: L2V2 Section~4.14.2; \changed{L2V3 Section~\ref{sec:eventassignment}.})
