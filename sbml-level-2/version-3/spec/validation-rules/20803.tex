The \changed{identifier given as the value} of a \token{symbol}
\changed{attribute} in any \InitialAssignment definition cannot also appear
as the value of \changed{the} \token{variable} \changed{attribute} in an
\AssignmentRule.  \changed{In other words, a model} \changed{cannot
simultaneously define both an initial assignment and an assignment rule for
the same species,} \changed{compartment or parameter in a model.}
(References: L2V2 Section~4.10.4; \changed{L2V3
Section~\ref{sec:initialAssignment}.})
