The \token{units} in a \Parameter definition must be a value chosen from
among the following: a built-in unit (e.g., \val{substance}, \val{time},
etc.), the identifier of a \UnitDefinition in the model, or one of the
predefined base units in SBML (e.g., \val{litre}, \val{mole}, \val{metre},
etc.).  (References: L2V1 Section 4.7.3; L2V2 Section~4.9.3; \changed{L2V3
Section~\ref{sec:parameter-units}.})
