The types of values within \token{piecewise} operators should all
be consistent: the set of expressions that make up the first
arguments of the \token{piece} and \token{otherwise} operators
within the same \token{piecewise} operator should all return
values of the same type.  (References: L2V2 Section~3.5.8; \changed{L2V3
Section~\ref{sec:mathmltype}.})
