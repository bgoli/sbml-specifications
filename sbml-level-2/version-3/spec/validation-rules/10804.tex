The XHTML content inside a \token{notes} element can only take one of the
following general forms: (1) a complete XHTML document beginning with the
element \token{<html>} and ending with \token{</xhtml>}; (2) the ``body''
portion of a document beginning with the element \token{<body>} and ending
with \token{</body>}; or (3) XHTML content that is permitted within a
\token{<body>} ... \token{</body>} elements.  (References: L2V3
Section~\ref{sec:notes}.)
