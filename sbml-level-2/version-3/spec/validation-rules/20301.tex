The top-level element within \token{math} in a \FunctionDefinition must be
one and only one MathML \token{lambda} element or a MathML
\token{semantics} element containing one and only one \token{lambda}
element.  (References: L2V1 Section~4.3.2; L2V2 Section~4.3.2;
\changed{L2V3 Section~\ref{sec:function-definition-math}.})
