Redefinitions of the built-in unit \token{substance} must be based on the
units \token{mole}, \token{item}, \token{gram}, \token{kilogram}, or
\token{dimensionless}.  More formally, a \UnitDefinition for
\token{substance} must simplify to a single \Unit in \changed{which either
(a) the \token{kind} attribute has a value of \val{mole},
\val{item}, \val{gram}, or \val{kilogram}, and the} \changed{\token{exponent}
attribute has a value of \val{1}, or (b) the \token{kind} attribute has a
value of \val{dimensionless}} \changed{with any \token{exponent} attribute value.}
(References: L2V1 Section~4.4.3; L2V2 Section~4.4.3; \changed{L2V3
Section~\ref{sec:built-in-units}.})
