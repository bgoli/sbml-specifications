All MathML content in SBML must appear within a \token{math}
element, and the \token{math} element must be either explicitly or
implicitly in the XML namespace
\val{http://www.w3.org/1998/Math/MathML}.  (References: L2V2
Section~3.5; L2V3 Section~3.4; L2V4 Section~3.4; L2V5 Section~\ref{sec:formulas}.)
