The \token{sbml} container element must declare the XML Namespace
for SBML, and this declaration must be consistent with the values
of the \token{level} and \token{version} attributes on the 
\token{sbml} element.  (References: L2V2 Section~4.1;
L2V3 Section~4.1; L2V4 Section~4.1\changed{; L2V5 Section~\ref{sec:sbml}}.)
