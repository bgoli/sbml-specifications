Redefinitions of the \changed{predefined SBML} unit \token{length} must be based on the unit
\token{metre} or \token{dimensionless}.  More formally, a \UnitDefinition
for \token{length} must simplify to a single \Unit in which either (a) the
\token{kind} attribute has a value of \val{metre} and the
\token{exponent} attribute has a value of \val{1}, or (b) the
\token{kind} attribute has a value of \val{dimensionless} with
any \token{exponent} value.  (References: L2V1 Section 4.4.3; L2V2
Section~4.4.3; L2V3 Section~4.4.3; L2V4 Section~\ref{sec:predefined-units}.)
