Top-level elements within an \token{annotation} element cannot use any SBML
namespace, whether explicitly (by declaring the namespace to be one of the
URIs \val{http://www.sbml.org/sbml/level1},
\val{http://www.sbml.org/sbml/level2}, 
\val{http://www.sbml.org/sbml/level2/version2},\\
\val{http://www.sbml.org/sbml/level2/version3}, 
\val{http://www.sbml.org/sbml/level2/version4}, or
\changed{\val{http://www.sbml.org/sbml/level2/version5}}), or implicitly (by failing
to declare any namespace).  (References: L2V2 Section~3.3.3; L2V3
Section~3.2.4; L2V4 Section~3.2.4; L2V5 Section~\ref{sec:annotation-use}.)
