The \token{listOf\_} container elements in a \Model instance are optional,
but if present, the elements must not be empty.  Specifically, if
any of the following are present in a \Model, they must not be empty:\\
\token{listOfFunctionDefinitions},
\token{listOfUnitDefinitions},
\token{listOfCompartmentTypes},
\token{listOf\-Species\-Types},
\token{listOfCompartments}, 
\token{list\-Of\-Species},
\token{listOf\-Parameters},
\token{listOf\-Initial\-Assignments},\\
\token{list\-Of\-Rules},
\token{listOf\-Constraints},
\token{listOfReactions} and \token{listOfEvents}.  (References:
This is requirement stemming from the XML Schema used for SBML;
L2V3 Section~4.2; L2V4 Section~4.2; L2V5 Section~\ref{sec:model}.)
