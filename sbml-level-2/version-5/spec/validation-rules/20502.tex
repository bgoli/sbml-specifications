If a \Compartment definition has a \token{spatialDimensions} value
of \val{0}, then its \token{units} attribute must not be set. If the
compartment has no dimensions, then no units can be associated
with a non-existent size.  (References: L2V1 Section~4.5.4;
L2V2 Section~4.7.5; L2V3 Section~4.7.5; L2V4 Section~4.7.5\changed{; L2V5 Section~\ref{sec:compartment-units}}.)
