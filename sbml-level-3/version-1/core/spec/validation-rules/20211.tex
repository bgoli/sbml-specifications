A \Model object definition may have the following optional attributes:
\token{metaid}, \token{sboTerm}, \token{id}, \token{name},
\token{substanceUnits}, \token{timeUnits}, \token{volumeUnits},
\token{areaUnits}, \token{lengthUnits}, \token{extentUnits} and
\token{conversionFactor}.  No other attributes are permitted.  (References:
SBML L3V1 Section~\ref{sec:model}.)
