The value of the attribute \token{volumeUnits} on a \Model object should be
either the units \val{litre}, \val{dimensionless}, or the identifier of a
\UnitDefinition object based on these units or a unit derived from
\val{metre} (with an \token{exponent} of \val{3}).  (References: SBML L3V1
Section~\ref{sec:model-volumeUnits}.)
