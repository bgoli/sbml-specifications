The value of the attribute \token{units} on a \Compartment object having
\token{spatialDimensions} of \val{2} should be either \val{area},
\val{dimensionless}, or the identifier of a \UnitDefinition object based on
either \token{metre} (with \token{exponent} equal to \val{2}) or
\token{dimensionless}.  (References: SBML L3V1
Section~\ref{sec:compartment-units}.)
