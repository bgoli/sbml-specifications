A \Reaction object must have the required attributes \token{id},
\token{reversible} and \token{fast}, and may have the optional attributes
\token{metaid}, \token{sboTerm}, \token{name} and \token{compartment}.  No
other attributes from the SBML Level~3 Core namespace are permitted on a
\Reaction object.  (References: SBML L3V1 Section~\ref{sec:reactions}.)
