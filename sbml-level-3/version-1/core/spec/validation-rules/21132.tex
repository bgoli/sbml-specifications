A \KineticLaw object must have the required attribute \token{id} and may
have the optional attributes \token{metaid}, \token{sboTerm}, \token{name},
\token{value} and \token{units}.  No other attributes from the SBML Level~3
Core namespace are permitted on a \KineticLaw.  (References: SBML L3V1
Section~\ref{sec:reactions}.)
