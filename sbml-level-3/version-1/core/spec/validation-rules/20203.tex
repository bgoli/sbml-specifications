The \token{listOf\_} container elements in a \Model instance are optional,
but if present, such container elements must not be empty.  Specifically,
if any of the following is present in a \Model, it must not be empty:
\token{listOfFunctionDefinitions}, \token{listOfUnitDefinitions},
\token{listOfCompartments}, \token{listOfSpecies},
\token{listOfParameters}, \token{listOfInitialAssignments},
\token{listOfRules}, \token{listOfConstraints}, \token{listOfReactions}
and \token{listOfEvents}.  (References: L2V3 Section~4.2; L2V4 Section~4.2;
L3V1 Section~\ref{sec:model}.)
