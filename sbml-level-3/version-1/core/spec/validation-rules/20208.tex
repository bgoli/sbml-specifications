The value of the \token{areaUnits} attribute on a \Model object definition
should be either \val{dimensionless} or the identifier of a \UnitDefinition
object based on \val{dimensionless} or a unit derived from \val{metre}
(with an \token{exponent} of \val{2}).  (References:
SBML L3V1 Section~\ref{sec:model-areaUnits}.)
