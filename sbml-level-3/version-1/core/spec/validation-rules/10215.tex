Outside of a \FunctionDefinition, if a \token{ci} element is not
the first element within a MathML \token{apply}, then the
\token{ci}'s value can only be chosen from the set of identifiers
of \Species, \Compartment, \Parameter or \Reaction objects defined
in the SBML model.  (References: L2V2 Section~4.3.2; L2V3
Section~3.4.3; L2V4 Section~3.4.3; L3V1 Section~\ref{sec:ci-token}.)
