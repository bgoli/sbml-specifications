The value of the \token{sboTerm} attribute on a \Model should be
an SBO identifier referring to \changed{an interaction} framework defined
in SBO.  That is, the value \changed{should} be a term derived from
\changed{\sbointeractionID}, \changed{``interaction''}.  (References: L2V2
Section~4.2.1; L2V3 Section~4.2.2; L2V4 Section~\ref{sec:model-sboterm}.)
