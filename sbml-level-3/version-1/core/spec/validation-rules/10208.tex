MathML \token{lambda} elements are only permitted as the first element
inside the \token{math} element of a \FunctionDefinition or as the
first element of a \token{semantics} element immediately inside the
\token{math} element of a \FunctionDefinition; they may not be used
elsewhere in an SBML model.  (References: SBML 
L3V1 Sections~\ref{sec:mathmlsubset} and~\ref{sec:function-definition-math}.)
