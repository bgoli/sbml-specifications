An \Event object may have the optional attributes \token{metaid},
\token{sboTerm}, \token{id}, \token{name}, and in addition, the
contingently-optional attribute \token{useValuesFromTriggerTime}.  No other
attributes from the SBML Level~3 Core namespace are permitted on an \Event
object.  (References: SBML L3V1 Section~\ref{sec:events}.)
