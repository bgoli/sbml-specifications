The value of the \token{sboTerm} attribute on a \Parameter should be an SBO
identifier referring to a quantitative parameter.  That is, the value \changed{should}
be a term derived from \sboparameter.  (References: L2V2 Section~4.9.5; L2V3
Section~4.9.5; L2V4 Section~4.9.5; L3V1 Section~\ref{sec:parameter-sboterm}.)
