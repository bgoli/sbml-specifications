A \Parameter object must have the required attributes \token{id} and
\token{constant}, and may have the optional attributes \token{metaid},
\token{sboTerm}, \token{name}, \token{value} and \token{units}.  No other
attributes from the SBML Level~3 Core namespace are permitted on a
\Parameter object.  (References: SBML L3V1 Section~\ref{sec:parameters}.)
