The syntax of unit identifiers (i.e., the values of the \token{id}
attribute on \UnitDefinition, the \token{units} attribute on \Compartment,
the \token{units} attribute on \Parameter, the \token{substanceUnits}
attribute on \Species, and the SBML \token{units} attribute on MathML
\token{ci} elements) must conform to the syntax of the SBML type
\primtype{UnitSId}.  (References: SBML L3V1 Section~\ref{sec:unitsid}.)
