The value of the \token{units} attribute on a \Compartment object having
\token{spatialDimensions} of \val{3} should be either \val{volume},
\val{litre}, or the identifier of a \UnitDefinition object based on either
\token{litre}, \token{metre} (with \token{exponent} equal to \val{3}), or
\token{dimensionless}.  (References: SBML L3V1
Section~\ref{sec:compartment-units}.)
