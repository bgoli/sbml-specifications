The value of a \Species's \token{substanceUnits} attribute can only be one of
the following: \val{substance}, \val{mole}, \val{item}, \val{gram},
\val{kilogram}, \val{dimensionless}, or the identifier of a
\UnitDefinition derived from \val{mole} (with an \token{exponent} of
\val{1}), \val{item} (with an \token{exponent} of \val{1}), \val{gram}
(with an \token{exponent} of \val{1}), \val{kilogram} (with an
\token{exponent} of \val{1}), or \val{dimensionless}.  (References: L2V1
Section~4.6.4; L2V2 Section~4.8.5; L2V3 Section~4.8.5; L2V4 Section~4.8.5; 
L3V1 Section~\ref{sec:species-units}.)
