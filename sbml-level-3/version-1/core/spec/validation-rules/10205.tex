In SBML Level~3, the only values permitted for the
\token{definitionURL} attribute on a \token{csymbol} are
{\scalefont{0.98}\val{http://www.sbml.org/sbml/symbols/time}},
{\scalefont{0.98}\val{http://www.sbml.org/sbml/symbols/delay}} and
{\scalefont{0.98}\val{http://www.sbml.org/sbml/symbols/avogadro}}.
(References: SBML L3V1 Section~\ref{sec:csymbol-token}.)
