A \Species object must have the following required attributes: 
\token{id}, \token{compartment}, \token{hasOnlySubstanceUnits},
\token{boundaryCondition} and \token{constant}
and may have the following optional attributes:
\token{metaid}, \token{sboTerm}, \token{name}, \token{initialAmount},
\token{initialConcentration}, \token{substanceUnits} and 
\token{conversionFactor}.  No other attributes are permitted.  
(References: SBML L3V1 Section~\ref{sec:species}.)
