If the \token{substanceUnits} attribute on a \Species object has not been
set, then the units of the species' quantity are determined by the value of
the \Model's \token{substanceUnits} attribute.  If neither the \Species
\token{substanceUnits} and \Model \token{substanceUnits} are set, then the
units of quantity of that species are undefined.  (References: L3V1
Section~\ref{sec:species-units}.)
