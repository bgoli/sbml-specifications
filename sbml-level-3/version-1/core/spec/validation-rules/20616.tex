If the attribute \token{substanceUnits} in a \Species object has not been
set, then the units of the species' quantity are determined by the value of
the enclosing \Model object's \token{substanceUnits} attribute.  If neither
the \Species object's \token{substanceUnits} attribute nor the enclosing
\Model object's \token{substanceUnits} attribute are set, then the units of
quantity of that species are undefined.  (References: SBML L3V1
Section~\ref{sec:species-units}.)
