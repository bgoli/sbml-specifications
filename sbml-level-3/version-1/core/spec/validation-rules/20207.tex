The value of the \token{volumeUnits} attribute on a \Model object
definition should be either \val{litre}, \val{dimensionless} or the
identifier of a \UnitDefinition object based on these units or a unit
derived from \val{metre} (with an \token{exponent} of \val{3}).
(References: SBML L3V1 Section~\ref{sec:model-volumeUnits}.)
