The value of the attribute \token{substanceUnits} on a \Model object should be
either the units \val{mole}, \val{item}, \val{avogadro},
\val{dimensionless}, \val{kilogram}, \val{gram}, or the identifier of a
\UnitDefinition object based on these units.  (References: \tworefs{4.2.3}{\ref{sec:model-substanceUnits}}.)
