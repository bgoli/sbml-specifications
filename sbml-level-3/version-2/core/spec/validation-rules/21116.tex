A \SpeciesReference object must have the required attributes
\token{species} and \token{constant}, and may have the optional attributes
\token{metaid}, \token{sboTerm}, \token{id}, \token{name} and
\token{stoichiometry}.  No other attributes from the SBML Level~3 Core
namespace are permitted on a \SpeciesReference object.  (References: SBML
L3V2 Section~\ref{sec:reactions}.)
