The value of the attribute \token{areaUnits} on a \Model object should be
either \val{dimensionless} or the identifier of a \UnitDefinition object
based on \val{dimensionless} or a unit derived from \val{metre} (with an
\token{exponent} of \val{2}).  (References: SBML L3V1 Section 4.2.5; SBML L3V2
\sec{sec:model-areaUnits}.)
