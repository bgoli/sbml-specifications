A \Species object must have the required attributes \token{id},
\token{compartment}, \token{hasOnlySubstanceUnits},
\token{boundaryCondition} and \token{constant}, and may have the optional
attributes \token{metaid}, \token{sboTerm}, \token{name},
\token{initialAmount}, \token{initialConcentration}, \token{substanceUnits}
and \token{conversionFactor}.  No other attributes from the SBML Level~3
Core namespace are permitted on a \Species object.  (References: \tworefs{4.6}{\ref{sec:species}}.)
