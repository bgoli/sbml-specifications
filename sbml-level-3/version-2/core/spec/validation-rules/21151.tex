A \ListOfModifierSpeciesReferences object may have the optional attributes
\vertwo{\token{id}, \token{name}, }\token{metaid} and \token{sboTerm}.  No other attributes from the SBML
Level~3 Core namespace are permitted on an object of class
\ListOfModifierSpeciesReferences.  (References: \tworefs{4.11}{\ref{sec:reactions}}.)
