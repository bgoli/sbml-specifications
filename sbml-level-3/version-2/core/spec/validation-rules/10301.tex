The value of the attribute \token{id} on every instance of the following
classes of objects must be unique across the set of all \token{id}
attribute values of all such objects in a model: \Model,
\FunctionDefinition, {\scalefont{0.98}\Compartment, \Species, \Reaction,
\SpeciesReference, \ModifierSpeciesReference, \Event, and \Parameter}.
(References: SBML L3V1 Section~\ref{sec:idnameattribs}.)
