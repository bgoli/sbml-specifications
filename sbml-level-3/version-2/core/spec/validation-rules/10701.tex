The value of the attribute \token{sboTerm} on a \Model object should be an
SBO identifier referring to an interaction \changed{or modeling} framework defined in SBO.  That
is, the value should be a term derived from \sbointeractionID,
``\changed{occurring entity representation}'', \changed{or \sboframeworkID, ``modeling framework''}.  (References: SBML L3V1 Section 5; SBML L3V2 \sec{sec:sbo}.)
