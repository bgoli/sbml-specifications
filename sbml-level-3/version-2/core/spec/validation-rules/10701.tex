The value of the attribute \token{sboTerm} on a \Model object should be an
SBO identifier referring to an interaction \relchange{or modeling} framework defined in SBO.  That
is, the value should be a term derived from \sbointeractionID,
``\relchange{occurring entity representation}'', \relchange{or \sboframeworkID, ``modeling framework''}.  (References: \tworefs{5}{\ref{sec:sbo}}.)
