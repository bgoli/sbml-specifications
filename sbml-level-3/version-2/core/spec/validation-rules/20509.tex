The value of the attribute \token{units} on a \Compartment object having
\token{spatialDimensions} of \val{3} should be either 
\val{litre}, or the identifier of a \UnitDefinition object based on either
\token{litre}, \token{metre} (with \token{exponent} equal to \val{3}), or
\token{dimensionless}, or a combination of these units.  (References: \tworefs{4.5.4}{\ref{sec:compartment-units}}.)
