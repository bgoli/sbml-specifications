A \Trigger object must have the required attributes
\token{persistent} and \token{initialValue}, and in addition, may have the
optional attributes \vertwo{\token{id}, \token{name}, }\token{metaid} and \token{sboTerm}.  No other
attributes from the SBML Level~3 Core namespace are permitted on a \Trigger
object.  (References: \tworefs{4.12}{\ref{sec:events}}.)
