In the SBML subset of MathML~2.0, the MathML attribute
\token{type} is only permitted on the \token{cn} construct.
No other MathML elements may have a \token{type} attribute.
\changed{An SBML package may allow the \token{type} attribute on other elements, and if so, the package must define \token{required}=\val{true} on the SBML container element \token{<sbml>}.}
(References: \tworefs{3.4.1}{\ref{sec:mathmlsubset}}.)
