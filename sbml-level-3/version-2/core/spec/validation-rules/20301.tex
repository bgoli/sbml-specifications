The top-level element within the MathML \token{math} element in a
\FunctionDefinition object, \vertwo{if present,} must be either exactly one MathML \token{lambda}
element, or exactly one MathML \token{semantics} element containing exactly
one \token{lambda} element.  (References: \tworefs{4.3.2}{\ref{sec:function-definition-math}}.)
