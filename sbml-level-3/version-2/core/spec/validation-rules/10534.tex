When the value of the attribute \token{variable} in a \RateRule object
refers to a \SpeciesReference object, the unit of measurement associated
with the \RateRule's \token{math} expression should be consistent with
\{unit derived from \token{dimensionless}\}/\{unit of \emph{time}\}.
(References: \tworefs{4.11.3, 4.2.4, and 4.9.4}{\ref{sec:speciesreferences-units}, \sec{sec:timeunits}, and~\sec{sec:raterule}}.)
