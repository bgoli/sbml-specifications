An \Event object must have the required attribute
\token{useValuesFromTriggerTime}, and in addition, may have the optional
attributes \token{metaid}, \token{sboTerm}, \token{id}, and \token{name}.
No other attributes from the SBML Level~3 Core namespace are permitted on
an \Event object.  (References: \tworefs{4.12}{\ref{sec:events}}.)
