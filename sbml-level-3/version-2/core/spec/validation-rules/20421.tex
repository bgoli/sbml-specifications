A \Unit object must have the required attributes \token{kind},
\token{exponent}, \token{scale} and \token{multiplier}, and may have the
optional attributes \vertwo{\token{id}, \token{name}, }\token{metaid} and \token{sboTerm}.  No other
attributes from the SBML Level~3 Core namespace are permitted on a \Unit
object.  (References: \tworefs{4.4}{\ref{sec:unitdefinitions}}.)
