The value of the attribute \token{extentUnits} on a \Model object should be
either the units \val{mole}, \val{item}, \val{avogadro},
\val{dimensionless}, \val{kilogram}, \val{gram}, or the identifier of a
\UnitDefinition object based on these units.  (References: SBML L3V2
\sec{sec:model-extentUnits}.)
