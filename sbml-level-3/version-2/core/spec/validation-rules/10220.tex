The SBML attribute \token{units} may only be added to MathML \token{cn}
elements; no other MathML elements are permitted to have the \token{units}
attribute.
\changed{An SBML Package may allow the \token{units} attribute in other places, and if so, the package must define \token{required}=\val{true} on the SBML container element \token{<sbml>}.}
(References: \tworefs{3.4.2}{\ref{sec:units-of-mathml}}.)
