The \token{id} attribute value of a \LocalParameter object defined within a
\KineticLaw object may only be used, \changed{in core}, in MathML \token{ci} elements within
the \token{math} element of that same \KineticLaw; in other words, the
identifier of the \LocalParameter object is not visible to other parts of
the model outside of that \Reaction instance.  \changed{In package constructs, the \token{id} attribute value of a \LocalParameter object may only be used in MathML \token{ci} elements or as the target of an \token{SIdRef} attribute if that package construct is a child of the parent \Reaction.}  (References: SBML L3V1 Sections 3.3.1, 3.4.3, and 4.11.5; SBML L3V2
\sec{sec:identifiers}, \sec{sec:ci-token},
and~\sec{subsec:kinetic-law}.)
