The value of a \Species object's \token{substanceUnits} attribute should
only be \relchange{one of the following\reltwo{\footnote{\changed{\val{substance} removed from this list, and proscribed exponents removed: any exponent now allowed.}}}:} \val{mole}, \val{item},
\val{gram}, \val{kilogram}, \val{dimensionless}, \val{avogadro} or the
identifier of a \UnitDefinition object derived from \relchange{\val{mole}, \val{item}, \val{gram}, \val{kilogram}, \val{avogadro}, or \val{dimensionless}, or a combination of these units}.
(References: \tworefs{4.6.5}{\ref{sec:species-substanceunits}}.)
