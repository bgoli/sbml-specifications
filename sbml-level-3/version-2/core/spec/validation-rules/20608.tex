The value of a \Species object's \token{substanceUnits} attribute should
only be \relchange{one of the following\reltwo{\footnote{\changed{\val{substance} removed from this list.}}}:} \val{mole}, \val{item},
\val{gram}, \val{kilogram}, \val{dimensionless}, \val{avogadro} or the
identifier of a \UnitDefinition object derived from \val{mole} (with an
\token{exponent} of \val{1}), \val{item} (with an \token{exponent} of
\val{1}), \val{gram} (with an \token{exponent} of \val{1}), \val{kilogram}
(with an \token{exponent} of \val{1}), \val{avogadro} (with an
\token{exponent} of \val{1}) or \val{dimensionless}.  (References: SBML L3V1 Section 4.6.5; SBML
L3V2 \sec{sec:species-substanceunits}.)
