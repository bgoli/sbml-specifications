An \AssignmentRule object must have the required attribute \token{variable}
and may have the optional attributes \changed{\token{id}, \token{name},} \token{metaid} and \token{sboTerm}.
No other attributes from the SBML Level~3 Core namespace are permitted on
an \AssignmentRule object.  (References: SBML L3V1
Section~\ref{sec:rules}.)
