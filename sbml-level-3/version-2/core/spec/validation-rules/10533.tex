When the value of the attribute \token{variable} in a \RateRule object
refers to a \Parameter object, the unit of measurement associated with
the \RateRule's \token{math} expression should be consistent with
\{parameter's \token{units}\}/\{unit of \emph{time}\}. (References: SBML
L3V2 \sec{sec:parameter-units}, \sec{sec:timeunits}
and~\sec{sec:raterule}.)
