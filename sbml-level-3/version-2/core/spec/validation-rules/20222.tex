A \Model object may only have the following attributes, all of which are
optional: \token{metaid}, \token{sboTerm}, \token{id}, \token{name},
\token{substanceUnits}, \token{timeUnits}, \token{volumeUnits},
\token{areaUnits}, \token{lengthUnits}, \token{extentUnits} and
\token{conversionFactor}.  No other attributes from the SBML Level~3 Core
namespace are permitted on a \Model object.  (References: \tworefs{4.2}{\ref{sec:model}}.)
