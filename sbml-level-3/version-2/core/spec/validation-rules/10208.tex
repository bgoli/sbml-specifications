MathML \token{lambda} elements are only permitted as either the first
element inside the \token{math} element of a \FunctionDefinition object, or
as the first element of a \token{semantics} element immediately inside the
\token{math} element of a \FunctionDefinition object.  MathML
\token{lambda} elements may not be used elsewhere in an SBML model.
\changed{An SBML package may allow \token{lambda} elements on other elements, and if so, the package must define \token{required}=\val{true} on the SBML container element \token{<sbml>}.}
(References: \tworefs{3.4.1 and 4.3.2}{\ref{sec:mathmlsubset} and \ref{sec:function-definition-math}}.)
