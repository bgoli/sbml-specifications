In the SBML subset of MathML~2.0, the MathML attribute
\token{definitionURL} is only permitted on \token{ci}, \token{csymbol} and
\token{semantics}.  No other MathML elements may have a
\token{definitionURL} attribute.
\changed{An SBML Package may allow the \token{definitionURL} attribute in other places, and if so, the package must define \token{required}=\val{true} on the SBML container element \token{<sbml>}.}
(References: \tworefs{3.4.1}{\ref{sec:mathmlsubset}}.)
