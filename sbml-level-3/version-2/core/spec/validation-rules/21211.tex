The value of the attribute \token{variable} in an \EventAssignment object
may only be the identifier of an existing \Compartment, \Species,
\Parameter or \SpeciesReference object in the modell, \changed{or any identifier in the SId namespace of the model belonging to an element defined by an SBML Level 3 package as having with mathematical meaning, and that is allowed to change over the course of a simulation.}  (References: \tworefs{4.12.5}{\ref{sec:eventassignment}}.)
