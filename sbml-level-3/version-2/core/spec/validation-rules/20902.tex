The value of a \RateRule object's \token{variable} attribute must be the
identifier of an existing \Species, \Compartment, \Parameter or
\SpeciesReference object defined in the model\vertwo{, or any identifier in the SId namespace of the model belonging to an element defined by an SBML Level 3 package as having with mathematical meaning, and that is allowed to change over the course of a simulation}.
