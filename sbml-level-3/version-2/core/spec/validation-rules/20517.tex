A \Compartment object must have the required attributes \token{id} and
\token{constant}, and may have the optional attributes \token{metaid},
\token{sboTerm}, \token{name}, \token{spatialDimensions}, \token{size} and
\token{units}.  No other attributes from the SBML Level~3 Core namespace
are permitted on a \Compartment object.  (References: SBML L3V2
Section~\ref{sec:compartments}.)
