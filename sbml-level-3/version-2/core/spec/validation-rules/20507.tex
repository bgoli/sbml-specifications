The value of the attribute \token{units} on a \Compartment object having
\token{spatialDimensions} of \val{1} should be either
\val{metre}, \val{dimensionless}, or the identifier of a \UnitDefinition
object based on either \token{metre} \changed{or
\token{dimensionless}}, or a combination of these units.  (References: \tworefs{4.5.4}{\ref{sec:compartment-units}}.)
