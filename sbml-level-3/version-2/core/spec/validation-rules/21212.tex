Any \Compartment, \Species, \Parameter or \SpeciesReference object whose
identifier is used as the value of the attribute \token{variable} of an
\EventAssignment object, must have a value of \val{false} for its
\token{constant} attribute.  (References: SBML L3V1 Section 4.12.5; SBML L3V2
\sec{sec:eventassignment}.)
