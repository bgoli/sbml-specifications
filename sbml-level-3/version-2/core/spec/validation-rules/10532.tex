When the value of the attribute \token{variable} in a \RateRule object
refers to a \Species object, the unit of measurement associated with the
\RateRule's \token{math} expression should be consistent with \{unit of
species quantity\}/\{unit of \emph{time}\}.  (References: SBML L3V1 Sections 4.6.5, 4.2.4, and 4.9.4; SBML L3V2
\sec{sec:species-units}, \sec{sec:timeunits}
and~\sec{sec:raterule}.)
