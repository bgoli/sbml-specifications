A \ModifierSpeciesReference object must have the required attribute
\token{species} and may have the optional attributes \token{metaid},
\token{sboTerm}, \token{id} and \token{name}.  No other attributes from the
SBML Level~3 Core namespace are permitted on a \ModifierSpeciesReference
object.  (References: SBML L3V2 \sec{sec:reactions}.)
