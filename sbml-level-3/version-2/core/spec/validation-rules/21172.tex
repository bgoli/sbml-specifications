A \LocalParameter object must have the required attribute \token{id} and
may have the optional attributes \token{id}, \token{name}, \token{metaid}, \token{sboTerm},
\token{name}, \token{value} and \token{units}.  No other attributes from
the SBML Level~3 Core namespace are permitted on a \LocalParameter object.
(References: \tworefs{4.11}{\ref{sec:reactions}}.)
