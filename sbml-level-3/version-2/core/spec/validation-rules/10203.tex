In the SBML subset of MathML~2.0, the MathML attribute
\token{encoding} is only permitted on \token{csymbol},
\token{annotation} and \token{annotation-xml}.  No other
MathML elements may have an \token{encoding} attribute.
\changed{An SBML Package may allow the \token{encoding} attribute in other places, and if so, the package must define \token{required}=\val{true} on the SBML container element \token{<sbml>}.}
(References: \tworefs{3.4.1}{\ref{sec:mathmlsubset}}.)
