If the attribute \token{substanceUnits} in a \Species object has not been
set, then the unit of measurement associated with the species' quantity is
determined by the value of the enclosing \Model object's
\token{substanceUnits} attribute.  If neither the \Species object's
\token{substanceUnits} attribute nor the enclosing \Model object's
\token{substanceUnits} attribute are set, then the unit of that species'
quantity is undefined.  (References: SBML L3V2
\sec{sec:species-units}.)
