An \InitialAssignment object must have the required attribute
\token{symbol} and may have the optional attributes \changed{\token{id}, \token{name},} \token{metaid} and
\token{sboTerm}.  No other attributes from the SBML Level~3 Core namespace
are permitted on an \InitialAssignment object.  (References: SBML L3V2
\sec{sec:initialAssignment}.)
