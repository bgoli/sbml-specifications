An \InitialAssignment object must have the required attribute
\token{symbol} and may have the optional attributes \vertwo{\token{id}, \token{name}, }\token{metaid} and
\token{sboTerm}.  No other attributes from the SBML Level~3 Core namespace
are permitted on an \InitialAssignment object.  (References: \tworefs{4.8}{\ref{sec:initialAssignment}}.)
